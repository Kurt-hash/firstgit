Configuring Git Repository

  1. Go to VS code and click Terminal. arrow then new terminal
  2. Check if git is already installed by typing git --version in the command line
  3. Configure git in your local computer
    • Set your username
      git config --global user.name "KurtOsorio"
    • Set your email address
      git config --global user.email "kurtjehu10@gmail.com
  4. To initialize the project directory type git init

Git add and Commit

  1. To see the status of the file in your directory type git status
  2. To track new files use the command git add
    Syntax git add. for group files
    Syntax git add file name for individual
  3. By running the git status again you can see the stage files in green ready to commit
  4. Commit the files that you have staged in your local repository by typing
    Syntax: git commit -m "added list in git document"

Git Branches



Git branch is a new version of repository. It allows you to work on your different parts of your project without impacting your main branch
  1. To list Branches type git branch
  2. To create a new branch git branch (branch name)
  3. To switch on a new branch git checkout (branch name)
